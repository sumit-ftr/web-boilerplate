# Project Setup

> Create `.env` file

> Set your **`INTERFACE IP`** and **`PORT`**
```
  SOCKET=<INTERFACE_IP>:<PORT>
```

> Paste your **`DATABASE URI`**
```
  DATABASE_URI=<your_database_uri>
```

> Set your **`DATABASE NAME`**
```
  DATABASE_NAME=<your_database_name>
```

> Set your **`SECRET KEY`**
```
  SECRET_KEY=<your_secret_key>
```

> Run your project
```
  cargo run --release
```
